Καλώς ήρθατε στο Sage¶

Αυτή είναι μία σύντομη περιήγηση στο Sage και στην χρήση του ως αριθμομηχανή.

Η γραμμή εντολών στο Sage εκκινεί με το μήνυμα προτροπής «sage:». Για πειραματισμό με τα ακόλουθα παραδείγματα, αρκεί να εισαγάγετε το μέρος μετά το μήνυμα προτροπής.

sage: 3 + 5
8

>>> from sage.all import *
>>> Integer(3) + Integer(5)
8

Εάν χρησιμοποιείτε το Sage σε σημειωματάριο Jupyter, τότε – παρομοίως – τοποθετείστε τα πάντα έπειτα του μηνύματος προτροπής εντός ενός κελιού εισαγωγής, και πατήστε Shift-Enter για να λάβετε την αντίστοιχη έξοδο.

Το σύμβολο εκθέτη σημαίνει «ύψωση σε δύναμη».

sage: 57.1^100
4.60904368661396e175

>>> from sage.all import *
>>> RealNumber('57.1')**Integer(100)
4.60904368661396e175

Υπολογίζουμε τον αντίστροφο ενός \(2 \times 2\) πίνακα στο Sage.

sage: matrix([[1, 2], [3, 4]])^(-1)
[  -2    1]
[ 3/2 -1/2]

>>> from sage.all import *
>>> matrix([[Integer(1), Integer(2)], [Integer(3), Integer(4)]])**(-Integer(1))
[  -2    1]
[ 3/2 -1/2]

Εδώ υπολογίζουμε το ολοκλήρωμα μίας απλής συνάρτησης.

sage: x = var('x')   # δημιουργίας συμβολικής μεταβλητής
sage: integrate(sqrt(x) * sqrt(1 + x), x)
1/4*((x + 1)^(3/2)/x^(3/2) + sqrt(x + 1)/sqrt(x))/((x + 1)^2/x^2 - 2*(x + 1)/x + 1)
- 1/8*log(sqrt(x + 1)/sqrt(x) + 1) + 1/8*log(sqrt(x + 1)/sqrt(x) - 1)

>>> from sage.all import *
>>> x = var('x')   # δημιουργίας συμβολικής μεταβλητής
>>> integrate(sqrt(x) * sqrt(Integer(1) + x), x)
1/4*((x + 1)^(3/2)/x^(3/2) + sqrt(x + 1)/sqrt(x))/((x + 1)^2/x^2 - 2*(x + 1)/x + 1)
- 1/8*log(sqrt(x + 1)/sqrt(x) + 1) + 1/8*log(sqrt(x + 1)/sqrt(x) - 1)

Εδώ το Sage καλείται να λύσει μία δευτεροβάθμια εξίσωση. Το σύμβολο == αντιπροσωπεύει την ισότητα στο Sage.

sage: a = var('a')
sage: S = solve(x^2 + x == a, x); S
[x == -1/2*sqrt(4*a + 1) - 1/2, x == 1/2*sqrt(4*a + 1) - 1/2]

>>> from sage.all import *
>>> a = var('a')
>>> S = solve(x**Integer(2) + x == a, x); S
[x == -1/2*sqrt(4*a + 1) - 1/2, x == 1/2*sqrt(4*a + 1) - 1/2]

Το αποτέλεσμα είναι μία λίστα από ισότητες.

sage: S[0].rhs()  # δεξί μέρος της εξίσωσης (rhs = right hand side)
-1/2*sqrt(4*a + 1) - 1/2

>>> from sage.all import *
>>> S[Integer(0)].rhs()  # δεξί μέρος της εξίσωσης (rhs = right hand side)
-1/2*sqrt(4*a + 1) - 1/2

Το Sage μπορεί να παραγάγει γραφήματα για διάφορες συναρτήσεις.

sage: show(plot(sin(x) + sin(1.6*x), 0, 40))

>>> from sage.all import *
>>> show(plot(sin(x) + sin(RealNumber('1.6')*x), Integer(0), Integer(40)))

Το Sage είναι μία πολύ ισχυρή αριθμομηχανή. Για να το δείτε αυτό, δημιουργείστε έναν \(500 \times 500\) πίνακα με τυχαίους αριθμούς.

sage: m = random_matrix(RDF, 500)

>>> from sage.all import *
>>> m = random_matrix(RDF, Integer(500))

Το Sage χρειάζεται ένα δευτερόλεπτο για τον υπολογισμό και την γραφική παρουσίαση των ιδιοτιμών του πίνακα.

sage: e = m.eigenvalues()  # περίπου 1 δευτερόλεπτο
sage: w = [(i, abs(e[i])) for i in range(len(e))]
sage: show(points(w))

>>> from sage.all import *
>>> e = m.eigenvalues()  # περίπου 1 δευτερόλεπτο
>>> w = [(i, abs(e[i])) for i in range(len(e))]
>>> show(points(w))

Το Sage χρειάζεται λιγότερο από 1 δευτερόλεπτο για να υπολογίσει τους τρόπους με τους οποίους ο αριθμός 100 εκατομμύρια μπορεί να γραφεί ως άθροισμα θετικών ακεραίων.

sage: z = Partitions(10^8).cardinality()  # περίπου .1 δευτερόλεπτα
sage: z
1760517045946249141360373894679135204009...

>>> from sage.all import *
>>> z = Partitions(Integer(10)**Integer(8)).cardinality()  # περίπου .1 δευτερόλεπτα
>>> z
1760517045946249141360373894679135204009...
